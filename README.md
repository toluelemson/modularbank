# modularbank

> A React application that opens a dummy contact form

## Technologies
* Parcel
* React
* Typescript
* Formik
* yup validator
* Material UI
* Eslint

## Install

```sh
npm install
```

## Usage

```sh
npm run start
```

## Build

```sh
npm run build
```

## Run Build Static

```sh
cd dist
```

```sh
npx serve
```

## Linting

```sh
npm run lint
```
